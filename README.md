### Gradle
1. Add this to `build.gradle` of project gradle dependency

```groovy
allprojects {
	repositories {
		...
 		maven { url 'https://jitpack.io' }
	}
}
```
2. Add this to `build.gradle` of app gradle dependency

```groovy
dependencies {
	implementation("com.github.jporigin:origin-ads:$version")
}
```
